Lab 02

Working with External Data (GADM)

Real-world GIS involves hunting for data. Learn to download, import, and visualize political hierarchies.

üéØ Learning Objectives

  • Locate and download authentic spatial data from a trusted external repository (GADM).
  • Import Shapefiles/GeoPackages into your GIS platform.
  • Understand administrative hierarchies (National vs. State vs. County).
  • Apply symbology rules (outline width, fill color) to visually distinguish hierarchy levels.

üìÇ Scenario & Sources

You are a country analyst for an NGO. You need to produce a clean reference map of a specific country's administrative divisions to include in a report. Instead of drawing it yourself, you will source the "Gold Standard" of administrative data: GADM (Database of Global Administrative Areas).

Primary Source: GADM.org

You will choose your own country to map.

Go to GADM Data →
Data Structure: GADM uses a level system:
  • ADM0: National Boundary (The Country)
  • ADM1: First Order Divison (States/Provinces/Regions)
  • ADM2: Second Order Division (Counties/Districts/Communes)

üõ†Ô∏è Step-by-Step Instructions

Select your preferred GIS platform:

1

Download Data

  1. Go to GADM Country Download.
  2. Select a country (suggestion: Ireland, Viet Nam, or Costa Rica‚Äîavoid huge countries like USA/Russia for now).
  3. Click "Shapefile" to download the ZIP.
  4. Important: Unzip the folder in your device's Documents/GIS/Lab02 folder.
2

Import Layers

Open ArcGIS Pro. Use the Catalog Pane to navigate to your Lab02 folder.

Drag and drop the _0 (National), _1 (State), and _2 (County) shapefiles onto the map.

Note: If you only see one shapefile, drag it in multiple times.

3

Symbolize Hierarchy

We want to see the boundaries clearly. Use the Symbology pane:

  • ADM2 (Bottom Layer): Set Fill to "No Color" (or very light gray). Set Outline Width to 0.5 pt (Gray).
  • ADM1 (Middle Layer): Set Fill to "No Color". Set Outline Width to 1.5 pt (Darker Gray).
  • ADM0 (Top Layer): Set Fill to "No Color". Set Outline Width to 3.0 pt (Black).

Arrange the draw order in the Contents pane so ADM0 is on top.

4

Layout & Export

Create a Layout (Insert → Layout). Add your map frame.

Add a Title ("Administrative Divisions of [Country]"), Scale Bar, North Arrow, and Legend.

Export as PDF.

1

Download Data

  1. Go to GADM Country Download.
  2. Select a country.
  3. Click "Geopackage" (Recommended for QGIS) or "Shapefile".
  4. Download and save to your project folder.
2

Load Data

Drag the Geopackage (.gpkg) into QGIS. It will ask which layers to add.

Select layers ending in 0, 1, and 2. Click OK.

3

Style the Hierarchy

Open Layer Properties → Symbology for each layer:

  • Layer 2 (Counties): Simple Fill → Transparent Fill. Stroke width: 0.1 mm.
  • Layer 1 (States): Simple Fill → Transparent Fill. Stroke width: 0.4 mm. Stroke Color: Dark Gray.
  • Layer 0 (Nation): Simple Fill → Transparent Fill. Stroke width: 0.8 mm. Stroke Color: Black.
4

Print Layout

Project → New Print Layout. Add Map. Add Legend.

Export as PDF.

1

Data Prep

ArcGIS Online requires zipped Shapefiles.

  1. Download the "Shapefile" version from GADM for a small country (e.g., Jamaica, Belgium). Large files will fail the free tier upload.
  2. Unzip the main download.
  3. Crucial Step: You must create a separate .zip file for each layer you want to upload.
    Example: Select gadm36_JAM_1.shp, .shx, .dbf, and .prj -> Right Click -> Send to Compressed Folder -> Name it Jamaica_Level1.zip.
2

Upload to Map Viewer

In Map Viewer, click Add → Add Layer from File.

Upload your Level0.zip, Level1.zip, and Level2.zip one by one.

Select "Import as a Feature Layer" if asked.

3

Visualize

Use the "Styles" pane (Shapes icon) on the right.

Set Fill transparency to 100%. Adjust outline widths to create the hierarchy effect (Thickest for Level 0, Thinnest for Level 2).

4

Print

Use the Print tool to generate a PDF of your map.
