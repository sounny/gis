Unlike raster analysis which treats the world as a continuous grid, vector analysis works with discrete objects (Points, Lines, Polygons). We manipulate the Geometry and Attributes of these objects to answer questions like:

  • "Which schools are within 1 mile of a toxic waste site?" (Buffer + Intersect)
  • "What is the land use variety inside each county?" (Union + Summary)
  • "How much of this property is in the flood zone?" (Clip)

üìç Proximity Analysis (Buffering)

Buffering creates polygons around features at a specified distance. It is the single most common vector analysis tool.

‚≠ï Fixed Distance

Every feature gets the same buffer size (e.g., "100 meters"). Useful for regulatory zones.

üî¢ Variable Distance

Buffer size depends on an attribute.
Ex: Large rivers = 500m buffer, Small creeks = 50m buffer.

üç© Multi-Ring

Multiple concentric buffers.
Ex: Hazard zones (High, Medium, Low risk based on distance).

üß© Overlay Operations

Overlay is the "Cookie Cutter" of GIS. It creates new features where layers overlap.

Operation Function Example Analogy
Clip Cuts Input to shape of Clip Layer. Attributes of Clip Layer are NOT kept. Cookie Cutter
Intersect Preserves only area common to BOTH inputs. Combines attributes. AND (A & B)
Union Preserves all features from BOTH inputs. Combines attributes. OR (A | B)
Erase Removes area that overlaps with Erase feature. NOT (A ! B)

üõ†Ô∏è Common Problems: Sliver Polygons

Warning: When you overlay data layers that don't perfectly line up (e.g., two different versions of a coastline), you get tiny, thin polygons called Slivers.

Fix: Use the "Integrate" tool or set XY Tolerance rules closer to your data precision.
